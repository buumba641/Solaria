# Solaria Android App

Create this project using Android Studio (Jetpack Compose).

## Required modules
- Solana Mobile Wallet Adapter (signing)
- Retrofit/OkHttp (backend)
- WalletConnect v2 (EVM signing for LI.FI routes)
- ExoPlayer / AudioTrack (voice playback)
- WebSocket client (ElevenLabs streaming)

## Planned screens
- Dashboard (balances + token perf)
- Voice overlay (mic streaming + responses)
- Transfer flow (prepare -> sign -> confirm)
- Cross-chain Top-up (LI.FI quote -> WalletConnect sign -> status polling)
- Merchant (payment link creation + intent settlement status)
